Beberapa format teks yang sering digunakan

1. Teks tebal menggunakan : "b dan /b"

2. Teks miring menggunakan : i dan /i

3. Menggaris bawahi teks menggunakan : u dan /u

4. Pangkat kecil dibawah teks menggunakan : sub dan /sub CONTOH : 20

5. Pangkat kecil diatas teks menggunakan : sup dan /sup CONTOH : 20

6. Teks menggunakan "em" CONTOH : Halo Onyet

7. Teks menggunakan "small" CONTOH : Halo Onyet

8. Teks menggunakan "strong" CONTOH : Halo Onyet

9. Teks menggunakan "ins" CONTOH : Halo Onyet

10. Teks menggunakan "del" CONTOH : Halo Onyet
